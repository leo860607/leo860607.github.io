個人化設置

github cornor(貓貓)

https://tholman.com/github-corners/
到上面網站選擇想要的樣式

  • 只放首頁
    放在themes/next/index.swig
  • 每一頁都要有
    放在themes/next/layout/_layout.swig

標籤與分類選項

  • 修改/theme/next/_config.yml
    打開tags和categories
  • 分別建立
    /source/categories/index.md
  • 以及/source/tags/index.md
  • 完成後重新佈署即可

網址列icon修改

  • 修改/theme/next/_config.yml
    把圖片放到\themes\next\source\images中
    確認圖片大小符合規定
